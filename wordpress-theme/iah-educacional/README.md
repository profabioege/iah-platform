# Tema WordPress — IAH Educacional

## Instalação

1. No painel WordPress, abra **Aparência > Temas**.
2. Selecione **Adicionar novo tema > Enviar tema**.
3. Envie o arquivo `iah-educacional-wordpress.zip` sem descompactá-lo.
4. Após a instalação, clique em **Ativar**.

O tema é uma home institucional estática, sem formulários, banco de dados, plugins ou dependências externas. Os links de navegação rolam para seções da página. O botão de demonstração é visual até que seja definido um canal de atendimento.
